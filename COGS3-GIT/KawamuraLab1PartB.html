Difference between text editor and word processor?
Text editors are used for creating and editing plain text files while word processors are used for creating and editing documents that require complex formatting
What is the client server-model?
The client-server model is a networking and computing architechture in which clients requests services or resources from central servers, enabling efficient dstribution of tasks and data in distributed systems
FTP
Description: File transfer protocol is a standard network protocol used for transferring files between a client and a server on a computer network Definition: FTP is a protocol that enables the exchange of files between a client and a server over a network, with defined commands for navigation and data transfer
HTTP
Description: Hypertext transfer protocol is the foundation of data communication on the World Wide Web. It is an application layer protocol used for transmitting and receiving web pages, images, videos, and other resources between web clients and web servers Definition: HTTP is a protocol that governs the transfer of hypertext (linked web documents) and other resources on the internet, specifying how requests and responses are formatted and transmitted
TCP/IP
Description: Transmission control protocol/Internet protocol is a suite of protocols that forms the backbone of the internet and many local area networks (LANs). It provides the foundation for data transmission and routing on the internet, ensuring that data packets are reliably delivered across networks Definition: TCP/IP is a comprehensive set of networking protocols that enable the reliable transmission of data across interconnected networks, including the internet, by dividing data into packets, addressing them, and ensuring their delivery to the intended destination
