How to define a table?
In HTML, you define a table using the <table> element, which serves as the container for your table.
Describe when to use the following HTML elements: <tr>, <th>, <td>, <thead>, <tfoot>, <caption>
Use <tr> to define individual rows, use <th> to deifne header cells in a table, use <td> to define data cells in a table, use <thead> to group the header rows in a table, use <tfoot> to group the footer rows in a table, use <caption> to add a caption to the table
What is the difference between colspan and rowspan?
colspan groups columns while rowspan groups rows of the table
What is the difference between cells spacing and padding?
Cell spacing in an HTML table refers to the gap between adjacent cells and is set with the cellspacing attribute, affecting the space between cells. Cell padding, set with the cellpadding attribute, controls the space within individual cells, creating space between cell content and cell borders. Both are used for layout and design, but they serve different purposes in table formatting.
How do you define table borders using CSS? How do you control border widths using CSS?
To define table borders using CSS, you can target the <table>, <th>, or <td> elements and use the border property. Set the border-collapse property to control how borders interact at cell boundaries. To control border widths, use the border-width property, specifying different widths in pixels, and you can also set the border style and color for the desired table elements.
